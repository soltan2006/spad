FAQ

What is the purpose of the project "SPADCHYNA - Belarus in Minecraft"?

Our project was created with the aim of popularizing and preserving the cultural heritage of Belarus through the virtual world of Minecraft.

What cultural heritage sites of Belarus can be seen in the project?

We have various objects of Belarusian culture, such as the Mir Castle Complex, Nesvizh Castle, Zhirovichi Monastery, the National Library in Minsk, Sovetskaya Square in Grodno, Mogilev City Hall, Brest Fortress, etc.

Where can I use the game worlds created in Minecraft?

The game worlds created in Minecraft can be used to create your own servers, to play online with friends, to hold themed competitions, and much more. There are practically no restrictions in using these worlds, only your imagination and creativity are important.

How are cultural objects created in Minecraft?

The objects are created based on the MCEdit program.
MCEdit is a program that allows you to edit maps, as well as everything on them. Highlighting is the main type of tool in MCEdit.
MCEdit helps you create real-size objects in a short time.

What version of Minecraft game worlds are running on?

Our maps work on version 1.12.2. It is important to remember that the game worlds are designed for the computer version of Minecraft!

How do I install game worlds?

To start the installation, find the card you need and unzip it to the folder if it is in the archive. Next, go to the following path:
c:\Users\Username\AppData\Roaming\.minecraft\saves\

And copy the folder with the map to the open saves folder. Now you can go to the single player tab in the game and find the installed map in the list of worlds. More detailed
