FAQ

Quel est le but du projet "SPADCHYNA - Biélorussie dans Minecraft"?

Notre projet a été créé dans le but de vulgariser et de préserver le patrimoine culturel de la Biélorussie à travers le monde virtuel de Minecraft.

Quels sites du patrimoine culturel de Biélorussie peuvent être vus dans le projet?

Nous avons divers objets de la culture biélorusse, tels que le complexe du château de Mir, le château de Nesvizh, le monastère de Zhirovichi, la Bibliothèque nationale de Minsk, la place Sovetskaya à Grodno, l'Hôtel de ville de Moguilev, la forteresse de Brest, etc.

Où puis-je utiliser les mondes de jeu créés dans Minecraft?

Les mondes de jeu créés dans Minecraft peuvent être utilisés pour créer vos propres serveurs, pour jouer en ligne avec des amis, pour organiser des compétitions thématiques, et bien plus encore. Il n'y a pratiquement aucune restriction dans l'utilisation de ces mondes, seuls votre imagination et votre créativité sont importantes.

Comment les objets culturels sont-ils créés dans Minecraft?

Les objets sont créés sur la base du programme MCEdit.
MCEdit est un programme qui vous permet d'éditer des cartes, ainsi que tout ce qui s'y trouve. La surbrillance est le principal type d'outil dans MCEdit.
MCEdit vous aide à créer des objets de taille réelle en peu de temps.

Sur quelle version des mondes de jeu Minecraft fonctionnent-ils?

Nos cartes fonctionnent sur la version 1.12.2. Il est important de se rappeler que les mondes du jeu sont conçus pour la version informatique de Minecraft!

Comment puis-je installer des mondes de jeu?

Pour démarrer l'installation, recherchez la carte dont vous avez besoin et décompressez-la dans le dossier si elle se trouve dans l'archive. Ensuite, allez au chemin suivant:
c:\Users\Username\AppData\Roaming\.minecraft\saves\

Et copiez le dossier avec la carte dans le dossier d'enregistrement ouvert. Vous pouvez maintenant accéder à l'onglet solo du jeu et trouver la carte installée dans la liste des mondes. Plus détaillé
